# Privacy Policy

*Last updated: 2026-07-16*

## Summary

xfs-forensic is a local Rust library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

xfs-forensic reads only the image bytes you pass to it. All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

xfs-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

xfs-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/xfs-forensic](https://github.com/SecurityRonin/xfs-forensic).
