# Changelog

All notable changes to `xfs-core` (the reader, imported as `xfs`) are documented
here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project adheres
to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.3]

- Current published reader: pure-Rust, `forbid(unsafe)`, panic-free-by-lint,
  input-fuzzed XFS parser — superblock, AG headers, inodes, extents, B-trees,
  directories, and v5 CRC32c self-describing-metadata verification.
